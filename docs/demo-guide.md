# StackSphere Presentation Demo Guide

Use this walkthrough structure when recording your final internship project video or presenting to evaluators.

---

## 📽️ Presentation Outline

### 1. Landing & Home Page UI (1 Minute)
* Show the main home screen (Stack Overflow style central Q&A feed).
* Show layout structure (Left navigation panel, central content, right widgets).
* Show layout responsiveness (resize browser window to show mobile columns).

### 2. User Onboarding & Logins (2 Minutes)
* **Registration:** Create a new test account (demonstrating validation errors).
* **Chrome Login (Security check):** Log in using Google Chrome. Show that email OTP is sent and required to proceed.
* **Edge Login (No-OTP check):** Log in using Microsoft Edge to demonstrate direct authentication bypass.
* **Mobile Login (Time restrictions):** Explain/demonstrate login blocking outside 10:00 AM - 1:00 PM.

### 3. User Profiles & Social Feed (2 Minutes)
* Go to Profile. Show login history records.
* **Posting Limits:**
  * Show a user with 0 friends trying to post (blocked).
  * Send a friend request and accept. Add 1 friend. Post once (succeeds), post twice (blocked).
  * Demonstrate media uploading (inserting an image/video to a post).
  * Interact with posts (likes, commenting).

### 4. Pricing & Subscriptions (2 Minutes)
* Open the subscription cards view.
* Attempt payment (sandbox mode) and explain the 10:00 AM - 11:00 AM IST transaction time restriction.
* Complete payment, show active plan update on profile.
* Check email inbox and show the automated PDF invoice delivery.

### 5. Reward System & Localization (2 Minutes)
* Open a Q&A post. Submit an answer to receive +5 points.
* Upvote the answer to 5 likes to show the extra +5 point bonus.
* Show the point transfer page. Transfer points to another user (validate user must have >10 points).
* Switch language to French (triggers email OTP challenge). Switch to Spanish/Hindi/others (triggers mobile OTP challenge).
